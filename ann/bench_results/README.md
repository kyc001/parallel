# Benchmark Results

Results are grouped by execution platform to avoid mixing or overwriting data from different machines.

- `android_termux_proot_ubuntu/`: Android tablet, Termux, PRoot Ubuntu, ARM AArch64 NEON.
- `kunpeng_server/`: reserved for the ARM Kunpeng server.
- `windows/`: reserved for Windows experiments.

Reusable benchmark programs and plotting scripts are in `../benchmarks/`.
